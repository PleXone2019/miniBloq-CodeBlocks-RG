      • Usada para SDKs de juegos
      • Usada para emuladores
      • Usada para demos
      • Usada para applicaciones multimedia

      • Eventos
      • Audio
      • CD-ROM de audio
      • Hilos
      • Temporizadores
      • Independencia del peso del sistema

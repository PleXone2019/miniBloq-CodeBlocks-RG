Detailed Description

This event class is used for the events generated by wxFilePickerCtrl and by wxDirPickerCtrl.

Events using this class

The following event handler macros redirect the events to member function handlers 'func' with prototypes like:

void handlerFuncName(wxFileDirPickerEvent& event)
Event macros:
  • EVT_FILEPICKER_CHANGED(id, func):
    Generated whenever the selected file changes.
  • EVT_DIRPICKER_CHANGED(id, func):
    Generated whenever the selected directory changes.

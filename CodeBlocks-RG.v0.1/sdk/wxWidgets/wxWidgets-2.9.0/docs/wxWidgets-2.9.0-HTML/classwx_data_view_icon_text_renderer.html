Detailed Description

The wxDataViewIconTextRenderer class is used to display text with a small icon next to it as it is typically done in a file manager.

This classes uses the wxDataViewIconText helper class to store its data. wxDataViewIonText can be converted to and from a wxVariant using the left shift operator.

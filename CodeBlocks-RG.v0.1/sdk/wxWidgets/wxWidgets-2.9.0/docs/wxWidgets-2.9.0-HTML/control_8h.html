enum wxEllipsizeFlags

Flags used by wxControl::Ellipsize function.

Enumerator:
wxELLIPSIZE_PROCESS_MNEMONICS  With this flag when calculating the size of the passed string, mnemonics characters (see wxControl::SetLabel) will be automatically reduced to a single character.

This leads to correct calculations only if the string passed to Ellipsize() will be used with wxControl::SetLabel. If you don't want ampersand to be interpreted as mnemonics (e.g. because you use wxControl::SetLabelText) then don't use this flag.

wxELLIPSIZE_EXPAND_TAB  This flag tells wxControl::Ellipsize to calculate the width of tab characters '\t' as 6 spaces.

wxELLIPSIZE_DEFAULT_FLAGS  The default flags for wxControl::Ellipsize.

enum wxEllipsizeMode

The different ellipsization modes supported by the wxControl::Ellipsize function.

Enumerator:
wxELLIPSIZE_START  Put the ellipsis at the start of the string, if the string needs ellipsization.
wxELLIPSIZE_MIDDLE  Put the ellipsis in the middle of the string, if the string needs ellipsization.
wxELLIPSIZE_END  Put the ellipsis at the end of the string, if the string needs ellipsization.

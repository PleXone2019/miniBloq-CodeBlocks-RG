Detailed Description

This event class contains information about a row/column resize event.

Events using this class

The following event handler macros redirect the events to member function handlers 'func' with prototypes like:

void handlerFuncName(wxGridSizeEvent& event)
Event macros:
  • EVT_GRID_COL_SIZE(func):
    The user resized a column by dragging it. Processes a wxEVT_GRID_COL_SIZE event type.
  • EVT_GRID_ROW_SIZE(func):
    The user resized a row by dragging it. Processes a wxEVT_GRID_ROW_SIZE event type.
  • EVT_GRID_CMD_COL_SIZE(id, func):
    The user resized a column by dragging it; variant taking a window identifier. Processes a wxEVT_GRID_COL_SIZE event type.
  • EVT_GRID_CMD_ROW_SIZE(id, func):
    The user resized a row by dragging it; variant taking a window identifier. Processes a wxEVT_GRID_ROW_SIZE event type.

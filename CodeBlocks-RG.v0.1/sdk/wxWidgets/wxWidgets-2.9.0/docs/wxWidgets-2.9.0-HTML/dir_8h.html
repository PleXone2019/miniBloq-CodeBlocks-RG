enum wxDirFlags

These flags define what kind of filenames are included in the list of files enumerated by wxDir::GetFirst() and wxDir::GetNext().

Enumerator:
wxDIR_FILES  Includes files.
wxDIR_DIRS  Includes directories.
wxDIR_HIDDEN  Includes hidden files.
wxDIR_DOTDOT  Includes "." and "..".
